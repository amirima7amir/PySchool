Урок 1: Переменные и типы данных

Что такое переменные?

Переменные — это имена, которые мы используем для хранения данных. В Python переменные создаются автоматически при присваивании значения.

name = "Анна"
age = 20
height = 1.68

Тип данных определяется автоматически. В примере выше:

  • name — строка (str)
  • age — целое число (int)
  • height — число с плавающей точкой (float)
