Урок 6: Списки и методы списков

Что такое списки?

Список в Python — это упорядоченная коллекция элементов, которая может содержать элементы разных типов.

my_list = [1, 2, 3, "яблоко", True]

Основные методы списков:

  • append() — добавляет элемент в конец списка
  • remove() — удаляет первый встреченный элемент
  • pop() — удаляет и возвращает элемент по индексу
  • sort() — сортирует список

Пример:

my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]
