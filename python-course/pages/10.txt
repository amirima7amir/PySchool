        <h2>Что такое множества?</h2>
        <p>Множества в Python — это неупорядоченные коллекции уникальных элементов. Множества не могут содержать дублирующиеся значения.</p>
        <pre><code>my_set = {1, 2, 3, 4}</code></pre>

        <h2>Основные операции с множествами:</h2>
        <ul>
            <li><code>add()</code> — добавляет элемент в множество</li>
            <li><code>remove()</code> — удаляет элемент из множества</li>
            <li><code>union()</code> — объединяет два множества</li>
            <li><code>intersection()</code> — находит пересечение двух множеств</li>
        </ul>

        <h3>Пример:</h3>
        <pre><code>my_set = {1, 2, 3}
my_set.add(4)
print(my_set)  # {1, 2, 3, 4}</code></pre>
