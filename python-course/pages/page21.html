Урок 20: Работа с библиотеками и пакетами в Python

Что такое библиотеки и пакеты?

Библиотеки и пакеты — это наборы готовых модулей, которые позволяют ускорить разработку. Библиотеки содержат функции и классы для решения различных задач, а пакеты — это коллекции таких библиотек.

Как установить библиотеку?

Для установки сторонних библиотек используется менеджер пакетов pip. Например, для установки библиотеки requests (для работы с HTTP-запросами) выполните команду:

pip install requests

Как использовать библиотеку?

После установки библиотеки, ее можно импортировать в программу с помощью команды import:

import requests

response = requests.get("https://api.github.com")
print(response.status_code)  # Код состояния HTTP-ответа

Создание собственного пакета:

Чтобы создать собственный пакет, нужно организовать структуру каталогов с файлами Python. Например:

my_package/
    _init_.py
    module1.py
    module2.py

Установка собственного пакета:

Для установки собственного пакета можно использовать команду pip install, указав путь к каталогу с файлом setup.py.

Основные библиотеки Python:

  • requests — работа с HTTP-запросами
  • numpy — работа с многомерными массивами и математическими функциями
  • pandas — работа с данными и таблицами
  • matplotlib — построение графиков и диаграмм
  • flask — создание веб-приложений

Пример использования библиотеки requests для отправки POST-запроса:

import requests

data = {'username': 'test', 'password': '1234'}
response = requests.post("https://httpbin.org/post", data=data)
print(response.json())  # Ответ в формате JSON

Пример использования pandas для работы с таблицами:

import pandas as pd

data = {'Имя': ['Аня', 'Петя'], 'Возраст': [25, 30]}
df = pd.DataFrame(data)
print(df)
