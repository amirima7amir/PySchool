        <h2>Цикл <code>for</code></h2>
        <p>Цикл <code>for</code> используется для перебора элементов в последовательности (список, строка и т.д.).</p>
        <pre><code>for i in range(5):
    print(i)</code></pre>
        <p>Результат: 0, 1, 2, 3, 4</p>

        <h2>Цикл <code>while</code></h2>
        <p>Цикл <code>while</code> выполняется, пока условие истинно:</p>
        <pre><code>i = 0
while i < 5:
    print(i)
    i += 1</code></pre>
        <p>Результат: 0, 1, 2, 3, 4</p>
