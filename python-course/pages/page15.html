Урок 14: ООП в Python

Что такое ООП?

Объектно-ориентированное программирование (ООП) — это парадигма программирования, основанная на концепции объектов, которые содержат данные и методы для их обработки.

Основные принципы ООП: инкапсуляция, наследование и полиморфизм.

Создание классов:

class Person:
    def _init_(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Привет, меня зовут {self.name} и мне {self.age} лет!")

# Создаем объект
person = Person("Анна", 25)
person.greet()

Основные элементы ООП:

  • class — создаем класс
  • _init_() — конструктор класса (метод инициализации)
  • self — ссылка на текущий объект
  • method — методы, которые определяются внутри класса

Наследование в Python:

class Student(Person):
    def _init_(self, name, age, school):
        super()._init_(name, age)
        self.school = school

    def greet(self):
        print(f"Привет, я {self.name}, мне {self.age} лет, учусь в {self.school}.")

student = Student("Иван", 18, "Школа №1")
student.greet()

Полиморфизм:

Полиморфизм позволяет использовать методы с одинаковыми именами, но разным поведением в разных классах.
