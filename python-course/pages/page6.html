Урок 5: Функции

Что такое функции?

Функции позволяют объединить набор инструкций в один блок, чтобы использовать его многократно.

def greet(name):
    print(f"Привет, {name}!")

greet("Анна")

Ключевые моменты:

  • def — ключевое слово для создания функции
  • name — параметр функции
  • Функцию можно вызывать сколько угодно раз
