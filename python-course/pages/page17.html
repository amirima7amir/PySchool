Урок 16: Работа с датами и временем в Python

Что такое модуль datetime?

Модуль datetime в Python позволяет работать с датами и временем. Он предоставляет классы для манипуляций с датами, временем и интервалами между ними.

Получение текущей даты и времени:

import datetime
now = datetime.datetime.now()
print(now)  # Например: 2025-04-28 14:32:01.123456

Создание объектов даты и времени:

birthday = datetime.datetime(1990, 5, 15)
print(birthday)  # 1990-05-15 00:00:00

Форматирование дат и времени:

Вы можете преобразовывать дату и время в строку с помощью метода strftime():

formatted = now.strftime("%d-%m-%Y %H:%M:%S")
print(formatted)  # 28-04-2025 14:32:01

Парсинг строки в объект даты и времени:

date_str = "28-04-2025"
date_obj = datetime.datetime.strptime(date_str, "%d-%m-%Y")
print(date_obj)  # 2025-04-28 00:00:00

Вычисления с датами и временем:

Для вычислений с датами можно использовать операторы, например, вычесть одну дату из другой, чтобы получить разницу:

today = datetime.datetime.now()
one_day = datetime.timedelta(days=1)
tomorrow = today + one_day
print(tomorrow)  # Например: 2025-04-29 14:32:01.123456

Работа с временными интервалами:

from datetime import timedelta
delta = timedelta(days=5)
print(delta)  # 5 days, 0:00:00
