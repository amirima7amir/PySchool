Урок 7: Строки и методы строк

Что такое строки?

Строки — это последовательности символов. В Python строки являются неизменяемыми объектами.

greeting = "Привет, мир!"

Основные методы строк:

  • upper() — переводит строку в верхний регистр
  • lower() — переводит строку в нижний регистр
  • split() — разделяет строку на части
  • replace() — заменяет часть строки

Пример:

text = "Привет, мир!"
print(text.upper())  # "ПРИВЕТ, МИР!"
