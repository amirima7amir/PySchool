Урок 11: Работа с файлами

Что такое файлы?

Файлы позволяют сохранять данные на диске. В Python можно работать с файлами, используя стандартные функции.

with open("example.txt", "w") as file:
    file.write("Привет, мир!")

Режимы работы с файлами:

  • "r" — открытие файла для чтения (по умолчанию)
  • "w" — открытие файла для записи (создается новый файл)
  • "a" — добавление данных в конец файла
  • "rb" — открытие файла для чтения в бинарном режиме

Пример:

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

Работа с ошибками при работе с файлами:

  • Если файл не существует, будет выброшено исключение FileNotFoundError
  • Использование конструкции try/except помогает обработать такие ошибки.
